devel_Up+

devel_Up+ is a serverless progressive web app built with React that allows users to view upcoming events in the world of web development from an integrated Google Calendar. Users are able to sign in with their Google account and access the app offline.

Background

This app was developed using a test-driven development approach. Working from user stories and Gherkin scenarios (see project README for more info), I created test suites for unit-testing, integration-testing, and end-to-end until I had fully functioning, error-free code. This project is built with a serverless architecture via AWS Lambda. Google OAuth 2.0 is used to access the calendar API for retrieving events. The app passes Lighthouse PWA checklist and users are able to install the app on their browser or mobile home screen. In addition, data visualization is included via recharts to provide users with an easily readable representation of the frequency, type, and location of calendar events.
