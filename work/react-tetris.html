Background

For this project, I set out to recreate the classic arcade game of Tetris. I considered using vanilla JavaScript at first, but then decided that it would be a fun challenge to try and build the game with React.js, allowing me to take advantage of the virtual DOM for quick, seamless rendering while getting some practice with functional components and custom hooks. Admittedly, I'm not typically a game developer and I wasn't quite sure if this would be the best choice, but upon completion of the app I have to say I'm quite pleased with how it came out- the game-play is smooth and responsive!
