Background

SocialApe was the first application I developed after finishing up my boot camp at CareerFoundry. Built from back to front entirely from scratch, it was quite a large undertaking.

I began by creating the backend code for the application. Working from user stories, I implemented the functionalities I wanted to include in the application (including creating posts, liking and unliking posts, creating profiles, sending notifications, and much more!) and tested all of my newly created endpoints via Postman.

Once all of my endpoints were functioning correctly and all of my data was being stored properly in my Google Firebase database, I moved on to the frontend of the application, which was built using React (initialized via CRA).

With the help of Material UI for webpage stylings, I implemented a sleek web design to display my user and scream data (SocialApe posts are called screams). The site includes login and signup pages, a homepage to display all screams (which you do not have to login to view), user profile pages, as well as modals for creating screams and viewing comments on individual screams.

Application states are managed via Redux

The site is currently live, hosted via Google Firebase!
