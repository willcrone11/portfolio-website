Star Wars Central

Star Wars Central is a full-stack web app that allows users to create an account and view information about their favorite Star Wars films! For this project, I used the same API to build two separate front ends, one built with React (MERN stack- MongoDB, Express.js, React, Node.js) and one built with Angular (MEAN stack).

Background

This app was built from front to back entirely from scratch and thus provided a great foray into both front-end and back-end development. The first step was to build the API, first creating the server and endpoints which I tested via Node.js and Postman, making sure to keep clean and informative documentation to allow others to use the API in the future. I then built the database for the application using MongoDB, eventually using Heroku to host the database online for use in the final application.

After the backend of the application was complete, I moved on to my front ends. The React version is compiled for production via Parcel and Redux is used for state management. Account security was also a priority in this app (as it always should be); all users who create an account have their passwords hashed and are assigned a user token that is used for authorization within the app.
