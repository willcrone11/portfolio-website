Chat-App

The Chat-App is a mobile app built using React-Native and Expo.io with an assist from the GiftedChat library. This single-room chat interface allows users to send messages, photos, and their current location. Messages and photos are stored via Google Firebase (NoSQL).

Background

This app was developed for mobile devices using Expo.io. Android Studio was used to test the app on various different kinds of mobile devices in addition to testing and development I did on my own iPhone. The design assets for the app were given at the start of the project and I built the code to meet these design specifications. The UI for the chat room was created with help the GiftedChat, allowing me to easily implement a very standard, recognizable, and intuitive chat UI that all users will likely be very familiar and comfortable with.

The app authenticates users anonymously via Google Firebase authentication and users are able to access the app while offline and see messages they have previously viewed (saved in local storage), although naturally they cannot send messages while offline. In addition to messages, users are also able to send their location, images from their photo libraries, as well as having the ability to take new photos with their camera to send within the app.
