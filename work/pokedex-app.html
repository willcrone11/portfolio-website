Pokédex App

The Pokédex App is a small web application developed as a front-end client that makes use of an external public API (PokéAPI) to return data about all 1050 Pokémon! The app was built with HTML, CSS, and object-oriented JavaScript, and also utilizes both JQuery and Bootstrap libraries.

Web App / External API / Fetch / AJAX
HTML / CSS / JavaScript / jQuery / Bootstrap

Background

This app was a learning project designed to explore the ins and out of object-oriented JavaScript with a focus on calling data from an external API and displaying the data in a clean, readable format. It was also a great opportunity to hone basic CSS and design skills, as well as to get practice with both the JQuery and Bootstrap libraries. In the development process user stories were used to layout the design elements and functionalities and elements were tested multiple times before putting them into production.
